# Health Fusion Landing

Marketing site for the Health Fusion mobile app concept.

This repository contains a polished landing page that translates the product story, visual language, and feature set of the app into a lightweight web experience. The site is built to present the product clearly rather than to house the mobile application itself.

## What this repo is

- A dedicated landing page for the Health Fusion app concept
- A product-storytelling layer for the broader app and design work
- A focused frontend project rather than a full backend product

## Product narrative

The current landing page presents Health Fusion as a health and wellness app centered on:

- AI-driven health predictions
- personalized daily guidance
- habit and activity tracking
- progress visualization
- a mobile-first user experience with app-download calls to action

The page structure includes:

- hero section with device mockup
- product stats
- feature grid
- how-it-works walkthrough
- testimonials
- FAQ
- footer / CTA sections

## Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion
- Lucide icons

## Project structure

- `app/`: Next.js app router entrypoints and global styles
- `components/`: landing page sections such as hero, features, testimonials, and FAQ
- `public/`: static brand assets and imagery

## Local development

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Why this repo stays in the portfolio

This project is intentionally narrower than a full product codebase, but it carries signal as a product-facing frontend artifact: it shows how the app concept is positioned, visualized, and presented on the web.
